## 1. OBJECTIVE

Currently, RED and BLUE agents are constrained to a **hardcoded action vocabulary** defined in `ActionType` enum (`cipher/agents/base_agent.py:34-76`). RED can only: move, read_file, exfiltrate, write/read dead_drop, wait, abort, and 4 predefined trap types. BLUE can only: investigate, trigger_alert, analyze_anomaly, reconstruct_path, stand_down, and 4 predefined trap types.

**This task removes those ceilings.** Agents must be able to **propose novel actions** not in the current enum — and the environment must evaluate, reward, or penalize them dynamically. The goal is to show judges that our agents *discover* strategies, not just pick from a menu.

---

## 2. WHAT "SELF-RELIANT" MEANS (DESIGN SPEC)

### 2.1 — Emergent Action Discovery

Agents should be able to output **free-text action proposals** that go beyond the predefined `ActionType` set. Examples of emergent behaviors we want the agents to discover on their own:

**RED Emergent Actions (not currently in ActionType):**
| Emergent Action | Description | How it should work |
|---|---|---|
| `spoof_credentials` | RED pretends to be an authorized admin | If on an `auth_gateway` node, reduce suspicion by 0.15. If on wrong node type, increase suspicion by 0.10 |
| `inject_noise` | RED floods the anomaly feed with junk signals | Reduce BLUE's `blue_detection_confidence` by 0.1 for 2 steps. Costs 1 step doing nothing else |
| `piggyback_service` | RED hides movement inside legitimate traffic | Move to adjacent node with 50% reduced suspicion cost. Only works on nodes with `http` or `ssh` services |
| `create_distraction` | RED triggers a fake alert on a distant node | BLUE agents' next observation shows a false anomaly at a random node in a different zone |
| `encrypt_exfil` | RED encrypts the file before exfiltrating | Exfiltration takes 2 steps instead of 1, but if successful, BLUE forensics cannot reconstruct the file name |
| `chain_dead_drops` | RED writes to multiple dead drops in sequence | Allows 3 drops in 1 step but each has 50% chance of corruption |

**BLUE Emergent Actions (not currently in ActionType):**
| Emergent Action | Description | How it should work |
|---|---|---|
| `network_quarantine` | BLUE isolates a subnet from the network | Remove all edges from a target node for 3 steps. Prevents RED movement through it |
| `deploy_canary_file` | BLUE places a fake classified file | If RED exfiltrates the canary, episode ends as BLUE win + RED gets -0.5 penalty |
| `traffic_analysis` | BLUE analyzes edge bandwidth patterns | Reveals which edges RED used in the last 3 steps (partial path exposure) |
| `credential_rotation` | BLUE changes auth tokens on a zone boundary | RED must re-authenticate, costing an extra step to cross zone boundaries |
| `shadow_agent` | BLUE surveillance mirrors RED's last known position | BLUE gets RED's exact node for 1 step, then position becomes stale |
| `set_alarm_tripwire` | BLUE attaches an invisible alarm to a specific edge | If RED traverses that edge, suspicion jumps by 0.25 instantly |

    # 1. Try standard ActionType matching (existing logic)

```python
def _extract_emergent_intent(self, raw: str, step: int) -> Action | None:
    """
    Parse free-text LLM output into an EmergentAction.
    Uses keyword heuristics to extract intent.
    """
    raw_lower = raw.lower()

    # Known emergent intent patterns
    EMERGENT_PATTERNS = {
        "spoof": "spoof_credentials",
        "inject noise": "inject_noise",
        "piggyback": "piggyback_service",
        "distraction": "create_distraction",
        "encrypt": "encrypt_exfil",
        "chain drop": "chain_dead_drops",
        "quarantine": "network_quarantine",
        "canary": "deploy_canary_file",
        "traffic analysis": "traffic_analysis",
        "credential rotation": "credential_rotation",
        "shadow": "shadow_agent",
        "tripwire": "set_alarm_tripwire",
        "alarm": "set_alarm_tripwire",
    }

    matched_intent = None
    for keyword, intent in EMERGENT_PATTERNS.items():
        if keyword in raw_lower:
            matched_intent = intent
            break

    if matched_intent is None:
        # If no known pattern, still allow it as a generic emergent action
        # but only if the response clearly proposes something novel
        if any(phrase in raw_lower for phrase in ["i want to", "attempt to", "try to", "propose"]):
            matched_intent = "custom_" + "_".join(raw_lower.split()[:4])
        else:
            return None

    # Extract target node from response (reuse existing regex)
    target_node = self._extract_target_node(raw)

    return Action(
        agent_id=self.agent_id,
        action_type=ActionType.EMERGENT,
        target_node=target_node,
        reasoning=raw[:300],
        step=step,
        emergent_data=EmergentAction(
            intent=matched_intent,
            target_node=target_node,
            reasoning=raw[:500],
            expected_effect="",
        ),
    )
```

### Step 3: Create the Emergent Action Evaluator

**File: `cipher/environment/emergent_evaluator.py`** (NEW FILE)

This is the core engine that evaluates emergent actions and applies state changes:

```python
"""
Emergent Action Evaluator for CIPHER.

Evaluates agent-proposed novel actions that aren't in the predefined
ActionType vocabulary. Each emergent intent has a context-dependent
effect on the episode state.
"""
from __future__ import annotations
from dataclasses import dataclass
from typing import Any, TYPE_CHECKING

if TYPE_CHECKING:
    from cipher.agents.base_agent import Action
    from cipher.environment.state import EpisodeState
    import networkx as nx

@dataclass
class EmergentResult:
    success: bool
    effect_description: str
    state_changes: dict[str, Any]
    reward_modifier: float  # Added to agent's reward
    suspicion_delta: float  # Change to suspicion score
    detection_delta: float  # Change to detection confidence
    penalty: float = 0.0   # Penalty if action was invalid/cheating

class EmergentEvaluator:
    """
    Evaluates emergent (novel) actions proposed by agents.

    Design philosophy: reward creative, context-appropriate actions.
    Penalize lazy or context-inappropriate ones.
    """

    # ── RED EMERGENT HANDLERS ────────────────────────────────────

    def _eval_spoof_credentials(self, action: Action, state: EpisodeState,
                                 graph: nx.Graph) -> EmergentResult:
        node = state.red_current_node
        node_type = graph.nodes[node].get("type", "")
        node_type_val = node_type.value if hasattr(node_type, "value") else str(node_type)

        if node_type_val in ("auth_gateway", "server"):
            return EmergentResult(
                success=True,
                effect_description="RED spoofed admin credentials — suspicion reduced",
                state_changes={"credential_level": "elevated"},
                reward_modifier=0.05,
                suspicion_delta=-0.15,
                detection_delta=0.0,
            )
        else:
            return EmergentResult(
                success=False,
                effect_description="Credential spoofing failed — wrong node type",
                state_changes={},
                reward_modifier=0.0,
                suspicion_delta=0.10,
                detection_delta=0.05,
            )

    def _eval_inject_noise(self, action: Action, state: EpisodeState,
                            graph: nx.Graph) -> EmergentResult:
        return EmergentResult(
            success=True,
            effect_description="RED injected noise into anomaly feed — BLUE confidence reduced",
            state_changes={"noise_injected_steps_remaining": 2},
            reward_modifier=0.03,
            suspicion_delta=0.0,
            detection_delta=-0.10,
        )

    def _eval_piggyback_service(self, action: Action, state: EpisodeState,
                                 graph: nx.Graph) -> EmergentResult:
        node = state.red_current_node
        services = graph.nodes[node].get("services", [])
        if any(s in services for s in ["http", "ssh", "https"]):
            return EmergentResult(
                success=True,
                effect_description="RED piggybacked on legitimate service traffic",
                state_changes={"piggybacked": True},
                reward_modifier=0.02,
                suspicion_delta=-0.08,  # Reduced suspicion for next move
                detection_delta=0.0,
            )
        return EmergentResult(
            success=False,
            effect_description="No piggybackable service on this node",
            state_changes={},
            reward_modifier=0.0,
            suspicion_delta=0.05,
            detection_delta=0.0,
        )

    def _eval_create_distraction(self, action: Action, state: EpisodeState,
                                  graph: nx.Graph) -> EmergentResult:
        import random
        # Pick a random node in a different zone than RED
        current_zone = graph.nodes[state.red_current_node].get("zone")
        distraction_candidates = [
            n for n in graph.nodes
            if graph.nodes[n].get("zone") != current_zone
        ]
        if distraction_candidates:
            distraction_node = random.choice(distraction_candidates)
            return EmergentResult(
                success=True,
                effect_description=f"RED triggered false alert at node {distraction_node}",
                state_changes={"distraction_node": distraction_node, "distraction_steps": 2},
                reward_modifier=0.04,
                suspicion_delta=-0.05,
                detection_delta=-0.05,
            )
        return EmergentResult(
            success=False,
            effect_description="Could not create distraction",
            state_changes={},
            reward_modifier=0.0,
            suspicion_delta=0.0,
            detection_delta=0.0,
        )

    # ── BLUE EMERGENT HANDLERS ───────────────────────────────────

    def _eval_network_quarantine(self, action: Action, state: EpisodeState,
                                  graph: nx.Graph) -> EmergentResult:
        target = action.target_node
        if target is None or target not in graph.nodes:
            return EmergentResult(False, "Invalid quarantine target", {}, 0.0, 0.0, 0.0)

        return EmergentResult(
            success=True,
            effect_description=f"BLUE quarantined node {target} — all edges removed for 3 steps",
            state_changes={"quarantined_node": target, "quarantine_steps": 3},
            reward_modifier=0.05,
            suspicion_delta=0.0,
            detection_delta=0.05,
        )

    def _eval_deploy_canary_file(self, action: Action, state: EpisodeState,
                                  graph: nx.Graph) -> EmergentResult:
        target = action.target_node
        if target is None:
            # Place canary on a random zone-2 node
            import random
            zone2_nodes = [n for n in graph.nodes if graph.nodes[n].get("zone") in (2, 3)]
            target = random.choice(zone2_nodes) if zone2_nodes else 0

        canary_name = f"canary_{target}_classified.docx"
        current_files = list(graph.nodes[target].get("files", []))
        current_files.append(canary_name)
        graph.nodes[target]["files"] = current_files

        return EmergentResult(
            success=True,
            effect_description=f"BLUE deployed canary file '{canary_name}' at node {target}",
            state_changes={"canary_file": canary_name, "canary_node": target},
            reward_modifier=0.03,
            suspicion_delta=0.0,
            detection_delta=0.0,
        )

    def _eval_traffic_analysis(self, action: Action, state: EpisodeState,
                                graph: nx.Graph) -> EmergentResult:
        path = list(getattr(state, "red_path_history", []))
        revealed = path[-3:] if len(path) >= 3 else path
        return EmergentResult(
            success=True,
            effect_description=f"BLUE analyzed traffic — revealed RED path: {revealed}",
            state_changes={"revealed_red_path": revealed},
            reward_modifier=0.04,
            suspicion_delta=0.0,
            detection_delta=0.08,
        )

    def _eval_set_alarm_tripwire(self, action: Action, state: EpisodeState,
                                  graph: nx.Graph) -> EmergentResult:
        target = action.target_node
        if target is None or target not in graph.nodes:
            return EmergentResult(False, "Invalid tripwire target", {}, 0.0, 0.0, 0.0)

        return EmergentResult(
            success=True,
            effect_description=f"BLUE set alarm tripwire on node {target}",
            state_changes={"tripwire_node": target},
            reward_modifier=0.02,
            suspicion_delta=0.0,
            detection_delta=0.0,
        )

    # ── GENERIC / UNKNOWN ────────────────────────────────────────

    def _eval_unknown(self, action: Action, state: EpisodeState,
                       graph: nx.Graph) -> EmergentResult:
        """
        Handle completely novel actions not in our registry.
        Small positive reward for creativity, but no state effect.
        """
        return EmergentResult(
            success=False,
            effect_description=f"Unknown emergent action: {action.emergent_data.intent if action.emergent_data else 'none'}",
            state_changes={},
            reward_modifier=0.01,  # Small creativity bonus
            suspicion_delta=0.0,
            detection_delta=0.0,
        )

    # ── MAIN DISPATCHER ──────────────────────────────────────────

    HANDLER_MAP = {
        "spoof_credentials": "_eval_spoof_credentials",
        "inject_noise": "_eval_inject_noise",
        "piggyback_service": "_eval_piggyback_service",
        "create_distraction": "_eval_create_distraction",
        "encrypt_exfil": "_eval_create_distraction",  # TODO: implement separately
        "chain_dead_drops": "_eval_create_distraction",  # TODO: implement separately
        "network_quarantine": "_eval_network_quarantine",
        "deploy_canary_file": "_eval_deploy_canary_file",
        "traffic_analysis": "_eval_traffic_analysis",
        "credential_rotation": "_eval_network_quarantine",  # Similar mechanic
        "shadow_agent": "_eval_traffic_analysis",  # Similar mechanic
        "set_alarm_tripwire": "_eval_set_alarm_tripwire",
    }

    def evaluate(self, action: Action, state: EpisodeState,
                  graph: Any) -> EmergentResult:
        if action.emergent_data is None:
            return self._eval_unknown(action, state, graph)

        intent = action.emergent_data.intent
        handler_name = self.HANDLER_MAP.get(intent)

        if handler_name and hasattr(self, handler_name):
            handler = getattr(self, handler_name)
            return handler(action, state, graph)

        return self._eval_unknown(action, state, graph)
```

### Step 4: Integrate Emergent Evaluator into Episode Runner

**File: `cipher/training/_episode_runner.py`**

In the `_execute_action()` or `_process_red_actions()` / `_process_blue_actions()` functions, add a branch for `ActionType.EMERGENT`:

```python
from cipher.environment.emergent_evaluator import EmergentEvaluator, EmergentResult

# Initialize once per episode
emergent_eval = EmergentEvaluator()

# In the action dispatch switch:
if action.action_type == ActionType.EMERGENT:
    result = emergent_eval.evaluate(action, state, graph)

    # Apply state changes
    if result.suspicion_delta != 0.0:
        state.red_suspicion_score = max(0.0, min(1.0,
            state.red_suspicion_score + result.suspicion_delta))
    if result.detection_delta != 0.0:
        state.blue_detection_confidence = max(0.0, min(1.0,
            state.blue_detection_confidence + result.detection_delta))

    # Store emergent state changes for downstream effects
    for key, value in result.state_changes.items():
        setattr(state, f"_emergent_{key}", value)

    # Log the emergent action
    state.episode_log.append({
        "step": current_step,
        "agent_id": action.agent_id,
        "action_type": "emergent",
        "payload": {
            "intent": action.emergent_data.intent if action.emergent_data else "unknown",
            "target_node": action.target_node,
            "success": result.success,
            "effect": result.effect_description,
        },
        "result": {
            "reward_modifier": result.reward_modifier,
            "suspicion_delta": result.suspicion_delta,
            "detection_delta": result.detection_delta,
        },
    })
```

def _check_emergent_abuse(self, episode_log: list[dict]) -> list[str]:
    """Flag agents that spam emergent actions or propose impossible things."""
    flags = []
    emergent_counts = {}
    for entry in episode_log:
        if entry.get("action_type") == "emergent":
            agent_id = entry.get("agent_id", "")
            emergent_counts[agent_id] = emergent_counts.get(agent_id, 0) + 1
            # Check for repeated failed emergent actions
            if not entry.get("result", {}).get("success", False):
                pass  # Accumulate failures

    for agent_id, count in emergent_counts.items():
        if count > 5:  # More than 5 emergent actions per episode = suspicious
            flags.append(f"EMERGENT_SPAM:{agent_id}:count={count}")

    return flags

## 6. IMPORTANT CONSTRAINTS

1. **Do NOT break existing ActionType parsing.** Emergent actions are a fallback — standard actions must still parse first.
2. **Do NOT allow emergent actions to bypass terminal conditions.** If suspicion hits 1.0, the episode still ends.
3. **Do NOT let emergent rewards exceed 0.3 total per episode.** Cap them to prevent reward hacking.
4. **ALL emergent actions MUST be logged** in `episode_log` for oversight auditing.
5. **OpenEnv compliance must still pass.** The `step()` API contract doesn't change — only the internal action space expands.
