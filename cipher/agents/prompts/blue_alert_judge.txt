- Prefer the node with the highest anomaly severity as the alert target.
- If honeypots_triggered_this_step is non-empty: lower threshold to 0.55.

- Never fire multiple alerts — pick the single highest-priority node.

OUTPUT FORMAT
Respond with ONLY a single JSON object:
